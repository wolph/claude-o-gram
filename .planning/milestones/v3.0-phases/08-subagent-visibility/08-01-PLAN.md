**2. Create src/monitoring/subagent-tracker.ts:**

Create a `SubagentTracker` class that manages all subagent lifecycle state. This is a standalone class with NO Telegram dependencies -- it only tracks state.

Export `ActiveAgent` interface:
```typescript
export interface ActiveAgent {
  agentId: string;
  agentType: string;       // "Explore", "Plan", etc.
  displayName: string;     // Human-friendly name for [bracket] prefix (use agentType)
  description: string;     // From PreToolUse(Agent) stash, or fallback to agentType
  startedAt: number;       // Date.now() for duration calculation
  depth: number;           // 0 = top-level subagent, 1 = nested, capped at 2
  parentAgentId?: string;  // undefined for top-level agents
  sessionId: string;       // Session this agent belongs to
}
```

Class `SubagentTracker` with methods:

- `stashDescription(sessionId: string, agentType: string, description: string): void` — Store description from PreToolUse(Agent) `tool_input.description`. Key by `${sessionId}:${agentType}`. Auto-expire entries after 30 seconds (setTimeout + delete).

- `popDescription(sessionId: string, agentType: string): string | null` — Retrieve and delete stashed description. Returns null if expired/missing.

- `start(sessionId: string, agentId: string, agentType: string): ActiveAgent` — Register a new active agent. Look up description via `popDescription`. Determine depth: if another agent is already active for this session, depth = parent.depth + 1 (capped at 2). Set parentAgentId if nested. Push agentId onto per-session stack (`sessionAgentStack: Map<string, string[]>`). Return the created `ActiveAgent`.

- `stop(agentId: string): { agent: ActiveAgent; durationMs: number } | null` — Remove agent from active map. Remove from session stack. Calculate duration. Return agent info + duration, or null if unknown agent.

- `getActiveAgent(sessionId: string): ActiveAgent | null` — Return the most recently started active agent for this session (top of stack). Used for temporal tool attribution.

- `getIndent(agentId: string): string` — Return indentation string: `"  "` (2 spaces) per depth level. Depth 0 = no indent, depth 1 = "  ", depth 2+ = "    ".

- `isSubagentActive(sessionId: string): boolean` — Quick check if any subagent is active for a session.

- `cleanup(sessionId: string): void` — Remove all agents and stashed descriptions for a session. Called on session end / /clear.

Internal state:
- `activeAgents: Map<string, ActiveAgent>` — keyed by agentId
- `sessionAgentStack: Map<string, string[]>` — per-session ordered stack of active agentIds
- `pendingDescriptions: Map<string, { description: string; timer: ReturnType<typeof setTimeout> }>` — keyed by `${sessionId}:${agentType}`

Discretion choices:
- Indent width: 2 spaces per depth level
- Batching: shared batcher (no per-agent batching -- keep it simple)
- Agent tracking: separate SubagentTracker class (not in SessionInfo)

/**
 * Format a subagent spawn announcement.
 * Per CONTEXT.md: inline text, symbol ▶, agent name, description.
 * indent = "  " per depth level (0 = no indent).
 */
export function formatSubagentSpawn(
  agentName: string,
  description: string,
  indent: string,
): string {
  const escapedName = escapeHtml(agentName);
  const escapedDesc = escapeHtml(truncateText(description, 150));
  const prefix = indent ? escapeHtml(indent) : '';
  return `${prefix}▶ <b>Agent(${escapedName})</b> spawned — "${escapedDesc}"`;
}
```

b) Export `formatSubagentDone`:

/**
 * Format a subagent completion announcement.
 * Per CONTEXT.md: inline text, symbol ✓, agent name, duration.
 * indent = "  " per depth level (0 = no indent).
 */
export function formatSubagentDone(
  agentName: string,
  durationMs: number,
  indent: string,
): string {
  const escapedName = escapeHtml(agentName);
  const durationStr = formatDuration(durationMs);
  const prefix = indent ? escapeHtml(indent) : '';
  return `${prefix}✓ <b>Agent(${escapedName})</b> done (${durationStr})`;
}
```
