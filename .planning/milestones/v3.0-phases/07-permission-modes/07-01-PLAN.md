**1. Create `src/control/permission-modes.ts`:**

Define and export:
```typescript
export type PermissionMode = 'manual' | 'accept-all' | 'same-tool' | 'safe-only' | 'until-done';

export interface ModeState {
  mode: PermissionMode;
  lockedToolName?: string;  // For 'same-tool' mode
  autoApprovedCount: number;
}
```

Create `PermissionModeManager` class:
- Private `modes = new Map<string, ModeState>()` keyed by sessionId.
- `getMode(sessionId: string): ModeState` -- returns stored state or `{ mode: 'manual', autoApprovedCount: 0 }` default.
- `setMode(sessionId: string, mode: PermissionMode, toolName?: string): void` -- creates entry. For 'same-tool', stores `lockedToolName`. Resets `autoApprovedCount` to 0.
- `resetToManual(sessionId: string): void` -- deletes entry from map (default is manual).
- `shouldAutoApprove(sessionId: string, toolName: string, toolInput: Record<string, unknown>): boolean`:
  1. If `isDangerous(toolName, toolInput)` returns true, return false (always prompt for dangerous commands regardless of mode).
  2. Switch on mode: manual=false, accept-all=true, same-tool=(toolName === lockedToolName), safe-only=(classifyRisk(toolName) === 'safe'), until-done=true.
- `incrementAutoApproved(sessionId: string): void` -- increments autoApprovedCount on the stored state.
- `cleanup(sessionId: string): void` -- deletes entry from map.

Create and export `isDangerous(toolName: string, toolInput: Record<string, unknown>): boolean`:
- Return false immediately if toolName is not 'Bash' and not 'BashBackground' (only shell commands can be dangerous).
- Extract `command` string from `toolInput.command`.
- Test against DANGEROUS_PATTERNS array of RegExps (per CONTEXT.md locked decision -- narrow blocklist):
  ```
  /\brm\s+(-[a-zA-Z]*f[a-zA-Z]*\s+|.*--force\s+).*\//  // rm -rf /
  /\bsudo\b/                                               // sudo anything
  /\bcurl\b.*\|\s*\bbash\b/                                // curl | bash
  /\bwget\b.*\|\s*\bbash\b/                                // wget | bash
  /\bgit\s+push\b.*--force\b/                              // git push --force
  /\bgit\s+push\b.*\s-f\b/                                 // git push -f
  ```
- Return true if any pattern matches.

Import `classifyRisk` from `../bot/formatter.js` for the safe-only mode check.

  ```typescript
  async handleStop(payload: StopPayload): Promise<void> {
    const session = this.sessionStore.get(payload.session_id);
    if (!session) return;
    this.sessionStore.updateLastActivity(payload.session_id);
    await this.callbacks.onStop(session, payload);
  }
  ```
