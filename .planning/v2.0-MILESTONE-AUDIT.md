## Tech Debt (3 items, no blockers)

### Documentation (Phase 5 scope)
- **README.md stale tmux references:** Lines 17, 125-150, 195 describe the deleted tmux architecture. Users reading the README will attempt tmux setup that is no longer needed.
- **.env.example missing ANTHROPIC_API_KEY:** New users following setup instructions will hit a fatal startup crash with no prior guidance.

### Code Hygiene (Phase 4 scope)
- **Dead config field:** `config.anthropicApiKey` is validated and stored in AppConfig but never read by any code path. The SDK reads `ANTHROPIC_API_KEY` directly from `process.env` at subprocess spawn time. No runtime impact.
